
# Linux for Cloud Engineers — Interactive Training Tool

## Live Demo

https://sergiugotacloud.github.io/linux-mastery-tool/

Interactive Linux training tool designed to improve real-world command usage, debugging skills, and interview readiness.

This is not a theoretical guide.  
It is built to simulate how Linux is actually used in cloud environments (EC2, logs, networking, processes).

---

## What this includes

1. Command Categories
- Navigation, files, permissions, processes, networking, system, pipes
- Practical examples for each command

2. Real Use Cases
- Log inspection
- Process debugging
- Network checks
- File system analysis

3. Interview Preparation
- Junior → Mid level questions
- Clear explanations with real context

4. Knowledge Check
- Built-in quiz system
- Reinforces retention through active recall

## Why I built this

Most Linux resources focus on memorization.

In real cloud environments, you need:
- fast navigation
- correct command selection
- ability to debug under pressure

This tool is built to reduce friction between:
problem → command → solution
